Detailed Description

Represents a chessboard. It is responsible for executing moves while containing the state of the game.

◆ move() [5/5]

MoveResult Chess::Board::move ( std::string_view  src,
std::string_view  destination 
)

Performs a move from a source to a destination and alternates between players according to the rules of chess. Coordinates are given column first and row second, from MIN_COLUMN to MAX_COLUMN and MIN_ROW to MAX_ROW, respectively. So, for instance, "C2" translates to 3rd column, 2nd row.

Returns an object containing information regarding the move executed. In case of invalid move, an InvalidMove exception is thrown.
